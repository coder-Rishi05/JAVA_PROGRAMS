Function 1: Find the Smallest Number

Question: Write a Java method to find the smallest number among three numbers.

Program Code

class Smallest {
  void smallestNumber(double a, double b, double c) {
    if (a < b && a < c) {
      System.out.println("A is the smallest number: " + a);
    } else if (b < a && b < c) {
      System.out.println("B is the smallest number: " + b);
    } else {
      System.out.println("C is the smallest number: " + c);
    }
  }
}

class Main {
  public static void main(String[] args) {
    Smallest obj = new Smallest();
    obj.smallestNumber(5, 9, 8);
  }
}
A is the smallest number: 5
