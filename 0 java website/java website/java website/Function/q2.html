Function 2: Average of Three Numbers

Question: Write a Java method to compute the average of three numbers.

Program Code

class Average {
  void averageNumber(double a, double b, double c) {
    System.out.println("Average of three numbers: " + (a + b + c) / 3);
  }
}

class Main {
  public static void main(String[] args) {
    Average obj = new Average();
    obj.averageNumber(3, 3, 3);
  }
}
Average of three numbers: 3.0
